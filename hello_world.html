TO DO: How to add/commit your work to Github.

Github is a reliable source for backing up and saving your code to a place where it can be referenced, anytime.

  • Create new file/edit existing file
  • anytime you add new work, make a commit
  • In IntelliJ, make a commit w/ command+k
  • Add instructions explaining what you're doing
  • Hover on "commit" button and select "commit and push"
